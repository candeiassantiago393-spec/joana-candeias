# Joana Candeias — Terapeuta Integrativa

Website profissional para [Joana Candeias](https://www.instagram.com/equilibriocomjoana_/), terapeuta integrativa em Costa da Caparica.

## Funcionalidades

- Apresentação dos serviços (massagens, pressoterapia, aulas individuais, nutrição)
- Formulário de marcações com redirecionamento para WhatsApp
- Botão flutuante de WhatsApp
- Links para Instagram profissional e pessoal
- Design responsivo com paleta verde marinho, creme e dourado

## Deploy no GitHub Pages

1. Cria um repositório no GitHub (ex: `joana-candeias`)
2. Faz push deste projeto para o repositório
3. Vai a **Settings → Pages**
4. Em **Source**, escolhe a branch `main` e pasta `/ (root)`
5. Guarda — o site ficará disponível em `https://<teu-user>.github.io/joana-candeias/`

## Desenvolvimento local

Abre o ficheiro `index.html` no browser, ou usa um servidor local:

```bash
python -m http.server 8080
```

Depois acede a `http://localhost:8080`
